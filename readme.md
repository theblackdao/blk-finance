# Blk Finance (\$BLKD)

## Overview

Official frontend repo of [blk.finance](https://app.blk.finance). Multichain yield farming optimizer.

## Env

Required env vars:

```
REACT_APP_NETWORK_ID = 56 // Network ID of BSC
```

## To Run

```
yarn
yarn start
```

The app will be running at http://localhost:3000/
